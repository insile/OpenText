##### JS.一元运算符
- JS.一元运算符
	- 一元运算符只有一个操作数
```js
A++
后置自增运算符

A--
后置自减运算符

++A
前置自增运算符

--A
前置自减运算符

delete
delete 运算符用来删除对象的属性。

void
void 运算符执行表达式并丢弃其返回值。

typeof
typeof 运算符用来判断给定对象的类型。

+
一元加运算符将操作数转换为 Number 类型。

-
一元减运算符将操作数转换为 Number 类型并取反。

~
按位非运算符。

!
逻辑非运算符。

await
暂停或恢复执行异步函数，并等待 promise 的兑现或拒绝。
```
