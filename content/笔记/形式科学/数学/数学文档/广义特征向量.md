##### 广义特征向量
- 广义特征向量
	- 设 $T$ 是[[向量空间|有限维向量空间]] $V$ 的[[线性算子]], $\lambda$ 是 $T$ 的[[特征值和特征向量|特征值]], 如果 $\pmb{v}\neq\pmb{0}$ 且对某个正整数 $k$ 有 $(T-\lambda I)^k\pmb{v}=\pmb{0}$, 则称向量 $\pmb{v}$ 是 $T$ 对应于 $\lambda$ 的广义特征向量. 并且非零向量 $\pmb{v}$ 是 $T$ 对应于 $\lambda$ 的广义特征向量, 当且仅当 $(T-\lambda I)^{\dim V}\pmb{v}=\pmb{0}$ . $T$ 对应于特征值 $\lambda$ 有[[广义特征空间]] $G(\lambda, T)$


>[!example]- 广义特征向量
>- $T \in L(\mathbb{C}^3)$, $T(z_1,z_2,z_3)=(4z_2,0,5z_3)$
>	- $T(\mathbf{e}_1) = T(1, 0, 0) = (4 \times 0, 0, 5 \times 0) = (0, 0, 0)$
>	- $T(\mathbf{e}_2) = T(0, 1, 0) = (4 \times 1, 0, 5 \times 0) = (4, 0, 0)$
>	- $T(\mathbf{e}_3) = T(0, 0, 1) = (4 \times 0, 0, 5 \times 1) = (0, 0, 5)$
>- $T$ 关于标准基的矩阵表示为 $T = \begin{bmatrix} 0 & 4 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 5 \end{bmatrix}$
>	- 特征值 $\lambda_1=0$ 特征向量 $z_1,0,0$
>	- 特征值 $\lambda_2=5$ 特征向量 $0,0,z_3$
