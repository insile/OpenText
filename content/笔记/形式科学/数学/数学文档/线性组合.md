##### 线性组合
- 线性组合
	- 线性组合是指对给定的一[[向量组]]进行标量乘法和向量加法运算得到的新向量 $\pmb{b}$, 即向量 $\pmb{b}$ 可由向量组线性表示, 它们 $(\pmb{a_1},\pmb{a_2},...,\pmb{a_n},\pmb{b})$ 是[[线性相关]]的, 可作为[[线性方程组的形式]], 本质就是[[线性变换]]. [[向量|零向量]] $\pmb{0}$ 是任意向量组的线性组合. 如果两个向量组可以互相线性表示，则称这两个向量组等价
		- $k_1\pmb{a_1}+k_2\pmb{a_2}+\cdots+k_n\pmb{a_n}=\pmb{b}$


>[!example]- 线性组合
> - $\pmb{a_1} = (1, 2)$, $\pmb{a_2} = (3, 4)$
> 	- $2\pmb{a_1} - \pmb{a_2} = (-1, 0)$
> - $\pmb{a_1} = (1, 2)$, $\pmb{a_2} = (2, 4)$
> 	- $2\pmb{a_1} + \pmb{a_2} = (4, 8)$
> - $\pmb{a}_1 = (1, 0, 0)$, $\pmb{a}_2 = (0, 1, 0)$, $\pmb{a}_3 = (0, 0, 1)$
> 	- $\pmb{a}_1+\pmb{a}_2+\pmb{a}_3=(1,1,1)$
> - $\pmb{a}_1 = (1, 2, 3)$, $\pmb{a}_2 = (2, 4, 6)$, $\pmb{a}_3 = (-1, 1, 0)$
> 	- $\pmb{a}_1+\pmb{a}_2+\pmb{a}_3=(2,7,9)$
