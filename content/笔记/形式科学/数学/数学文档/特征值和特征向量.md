##### 特征值和特征向量
- 特征值和特征向量
	- 设[[线性算子]] $T: V \to V$, 对于数 $\lambda\in\mathbb{F}$, 若存在非零向量 $\pmb{v}\in V$ 使得 $T\pmb{v}=\lambda\pmb{v}\iff (T-\lambda I)\pmb{v}=\pmb{0}$, 则称 $\lambda$ 为 $T$ 的**特征值**, 该向量 $\pmb{v}$ 是 $T$ 对应于特征值 $\lambda$ 的**特征向量**, 特征向量指明了变换中保持不变的方向, 特征值则表示这些方向上的缩放比例. $T$ 对应于特征值 $\lambda$ 有[[特征空间]] $E(\lambda, T)$ . [[广义特征向量]]是对特征向量的推广. $T$ 对应于特征值 $\lambda$ 还有[[广义特征空间]] $G(\lambda, T)$. 特征值具有[[特征值的重数|重数]]
- 特征值和特征向量性质与条件
	- 非零有限维复向量空间上的每个算子都有特征值
	- 特征值即最小多项式的零点


>[!example]- 特征值和特征向量
>- $T \in L(\mathbb{C}^3)$, $T(z_1,z_2,z_3)=(4z_2,0,5z_3)$
>	- $T(\mathbf{e}_1) = T(1, 0, 0) = (4 \times 0, 0, 5 \times 0) = (0, 0, 0)$
>	- $T(\mathbf{e}_2) = T(0, 1, 0) = (4 \times 1, 0, 5 \times 0) = (4, 0, 0)$
>	- $T(\mathbf{e}_3) = T(0, 0, 1) = (4 \times 0, 0, 5 \times 1) = (0, 0, 5)$
>- $T$ 关于标准基的矩阵表示为 $T = \begin{bmatrix} 0 & 4 & 0 \\ 0 & 0 & 0 \\ 0 & 0 & 5 \end{bmatrix}$
>	- 特征值 $\lambda_1=0$ 特征向量 $z_1,0,0$
>	- 特征值 $\lambda_2=5$ 特征向量 $0,0,z_3$
