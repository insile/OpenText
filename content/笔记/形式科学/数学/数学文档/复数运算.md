##### 复数运算
- 复数加法
	- 对于两个复数数 $z_1 = x_1 + y_1i$ 和 $z_2 = x_2 + y_2i$ ,其加法定义为 $z_1+z_2=(x_1+x_2,y_1+y_2)$
		- $z_1 + 0 = z_1$
		- $-z_1=(-x_1,-y_1)$
		- $z_1 + z_2 = z_2 + z_1$ 
		- $(z_1 + z_2) + z_3 = z_1 + (z_2 + z_3)$ 
- 复数乘法
	- 对于两个复数数 $z_1 = x_1 + y_1i$ 和 $z_2 = x_2 + y_2i$ ,其乘法定义为 $z_1z_2=(x_1x_2-y_1y_2,y_1x_2+x_1y_2)$ 
		- $z_1\cdot 1=z_1$
		- $z_1\cdot 0=0$
		- $\displaystyle z_1^{-1}=(\frac{x_1}{x_1^2+y_1^2},\frac{-y_1}{x_1^2+y_1^2})$
		- $z_1z_2 = z_2z_1$
		- $(z_1z_2)z_3 = z_1(z_2z_3)$
		- $z_3(z_1 + z_2)=z_3z_1+z_3z_2$
- 复数减法
	- 对于两个复数数 $z_1 = x_1 + y_1i$ 和 $z_2 = x_2 + y_2i$ , 其减法定义为 $z_1-z_2=z_1+(-z_2)$
- 复数除法
	- 对于两个复数数 $z_1 = x_1 + y_1i$ 和 $z_2 = x_2 + y_2i$ , 其除法定义为 $\displaystyle\frac{z_1}{z_2}=z_1z_1^{-1}$
