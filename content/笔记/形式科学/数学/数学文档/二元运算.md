##### 二元运算
- 二元运算
	- 设 $S$ 为一个非空集合，[[映射]] $f:S\times S\rightarrow S$ 称为 $S$ 上的二元运算也就是说，在非空集合S上任取两个元素 $a$, $b$ 做映射 $f$, 都将在该非空集合中唯一确定一个对应的元素 $c$, 即 $f(a,b)=c$, 记为 $a*b=c$
- 二元运算实例
	- 加法 $a+b$
		- $a + 0 = a$
		- $a + b = b + a$
		- $(a + b) + c = a + (b + c)$
	- 减法 $a-b=a+(-b)$
		- $a − b = −(b − a)$
	- 乘法 $a\cdot b$ 
		- $a\cdot 1=a$
		- $a\cdot 0=0$
		- $a\cdot b=b\cdot a$
		- $(a\cdot b)\cdot c=a\cdot (b\cdot c)$
		- $a\cdot (b+ c)=a\cdot b+a\cdot c$
	- 除法 $\displaystyle\frac{a}{b}=a\cdot\frac{1}{b}$ , $b\neq0$
	- 幂运算 $a^n=a\cdot a\cdot ...\cdot a$
		- $a^1=a$
		- $a^0=1$
		- $\displaystyle a^{-n}=\frac{1}{a^n}$
		- $\displaystyle a^{\frac{1}{n}}=\sqrt[n]{a}$
		- $a^{m+n}=a^m+a^n$
		- $(a^m)^n=a^{mn}$
		- $(a\cdot b)^n=a^n\cdot b^n$
		- $a^{m^n}=a^{(m^n)}$
	- 根式 $\sqrt[n]{a}$
		- $\displaystyle \sqrt[n]{a}=a^{\frac{1}{n}}$
		- $(\sqrt[n]{a})^n=a$
		- $\displaystyle\sqrt[n]{a^m}=(a^m)^{\frac{1}{n}}=a^{\frac{m}{n}}=(a^{\frac{1}{n}})^m=(\sqrt[n]{a})^m$
		- $\sqrt[n]{ab}=\sqrt[n]{a}\sqrt[n]{b}$
		- $\displaystyle\sqrt[n]{\frac{a}{b}}=\frac{\sqrt[n]{a}}{\sqrt[n]{b}}$
	- 对数 $\log_ab$
		- $a^{\log_ab}=b$
		- $\log_a(bc)=\log_ab+\log_ac$
		- $\displaystyle\log_a\frac{b}{c}=\log_ab-\log_ac$
		- $\log_a(b^c)=c\log_ab$
		- $\displaystyle\log_ab=\frac{\log_cb}{\log_ca}$
		- $\log_eb=\ln b$
		- $\log_{10}b=\lg b$
