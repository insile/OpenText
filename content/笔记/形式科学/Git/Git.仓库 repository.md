##### 仓库 repository
- 仓库 repository
	- 仓库是一个用于存储项目文件, 历史记录以及版本控制信息的文件夹. 仓库是 Git 操作的核心单位, 无论是个人项目还是团队协作项目, 所有的代码和相关信息都存储在仓库中. 项目文件夹被初始化为仓库时, Git 会创建一个名为 `.git` 的隐藏目录, 该目录包含所有的版本控制信息和配置文件
- 本地仓库 local repository
	- 本地仓库是存储在本地计算机上的仓库, 是在本地对代码进行编写以及 Git 操作的地方
- 远程仓库 remote repository
	- 远程仓库是存储在服务器上的仓库, 通常托管在平台如 GitHub, GitLab, Bitbucket 等, 如果不需要共享代码多人协作的话, 远程仓库就是多余的, 只需在本地仓库即可使用完整的 Git 操作进行版本控制
