 ##### Git 说明
- Git 说明
	- Git 是一个分布式版本控制系统, 用于跟踪项目文件的更改, 并允许多个开发人员同时协作. 本内容参考文档只对 Git v2.46.0 作简要说明, 详细术语, 命令, 参数, 底层原理等需查阅官网文档
- Git 官网与文档
	- https://git-scm.com/
	- https://git-scm.com/book/zh/v2
- Git GUI 
	- [GitHub Desktop](https://github.com/apps/desktop)是用于简化和增强本地代码管理流程的 Git 图形化客户端, 能简化开发工作流程
