##### 命令
- 命令
	- 仓库与分支管理
		- [[git init]] 仓库初始化
		- [[git clone]] 仓库克隆
		- [[git remote]] 远程仓库管理
		- [[git branch]] 分支管理
		- [[git switch]] 切换分支
		- [[git config]] Git 配置
	- 版本管理
		- 查
			- [[git status]] 版本状态
			- [[git log]] 提交历史
			- [[git diff]] 变化比较
		- 增
			- [[git add]] 添加暂存
			- [[git commit]] 提交版本
			- [[git merge]] 合并
			- [[git rebase]] 变基
			- [[git push]] 推送远程仓库
			- [[git fetch]] 更新
			- [[git pull]] 更新合并
		- 改
			- [[git reset]] 版本重置
			- [[git revert]] 版本回退
			- [[git restore]] 文件恢复
