##### 复合表达式
- 复合表达式
	- **复合表达式**是指包含多个[[C.运算符|运算符]]和​​[[C.操作数|操作数]]的复杂[[C.表达式|表达式]], 由多个子表达式组成, 每个子表达式可以是简单的或更复杂的, 操作的顺序由运算符的优先级和结合性决定, 可能涉及多种类型 (如整数, 浮点数, 指针, 数组) 的混合运算
```c
#include <stdio.h>

int main() {
	int a = 5, b = 10, c = 3;
	int result = ((a + b) * c - 7) / 2 + (a * b - c);  // 复合表达式
	printf("result = %d", result);
    return 0;
}

```
