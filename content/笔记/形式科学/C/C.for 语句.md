##### for 语句
- for 语句
	- 循环控制语句, 可以将语句或复合语句重复指定的次数, 在可选条件变为 false 前, for 语句的主体将执行零次或多次, 可以在 for 语句中使用可选表达式, 以便在 for 语句的执行期间初始化和更改值
- 语法
```c
for ( 初始化表达式 ; 条件表达式 ; 更新表达式 ) 循环语句
    // 执行初始化表达式
    // 计算条件表达式, 如果条件为真, 继续执行循环体; 如果为假, 跳出循环
    // 执行循环体
    // 执行更新表达式
    // 返回第 2 步, 重新计算条件表达式, 直到条件为假
```
- 示例
```c
// 打印数字 1-10
#include <stdio.h>

int main() {
    for (int i = 1; i <= 10; i++) {
        printf("%d\n", i);
    }
    return 0;
}

```
