##### 字符串字面量
- 字符串字面量
	- **字符串字面量**是指在 C 语言中直接表示字符串固定值的[[C.字面量|字面量]], 就是原位构造[[C.数组|字符数组]]类型, 字符串结尾隐式包含空字符 `\0`
- 语法
	- `"字符序列"`
	- `u8"字符序列"` 
	- `u"字符序列"` 
	- `U"字符序列"` 
	- `L"字符序列"` 
- 示例
```c
#include <stdio.h>

int main() {
    // 使用 u8 前缀定义 UTF-8 编码字符串
    char str1[] = u8"你好，世界";  // 这里的 str 是一个 char 数组
    char str2[] = "你好，世界";  // 这里的 str 是一个 char 数组

    printf("%s\n", str1);  // 使用 printf 输出字符串
    printf("%s\n", str2);  // 使用 printf 输出字符串
    return 0;
}

```
