##### 复合语句
- 复合语句
	- 复合语句也称为块语句, 是由一对花括号 `{}` 包含的一组语句. 复合语句将多个语句组合在一起, 使它们在需要单个语句的地方作为一个整体使用. 复合语句通常作为另一条语句的主体出现, 声明和类型描述可在复合语句的头部出现的声明的格式和含义
- 语法
```c
{
    语句1;
    语句2;
    // ...更多语句
}
```
- 示例
```c
// 在 if 语句中使用复合语句
#include <stdio.h>

int main() {
    int num = 10;

    if (num > 0) {
        printf("The number is positive.\n");
        printf("The number is %d.\n", num);
    }

    return 0;
}
```
