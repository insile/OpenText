##### 整数声明
- 整数声明
	- **整数声明**是使用整数类型说明符[[C.声明|声明]]一个[[C.整数|整数]]
- 语法
	- `int... variable_name;`
- 示例
```c
int x;    // 声明一个名为 x 的整数变量
int a, b, c;   // 声明三个整数变量 a, b 和 c
short small_num;  // 声明并初始化一个 short 类型的变量
long large_num;  // 声明并初始化一个 long 类型的变量
long long very_large_num;  // 声明并初始化一个 long long 类型的变量
unsigned int positive_num;  // 声明并初始化一个无符号整数变量
unsigned short small_positive_num;  // 声明并初始化一个无符号 short 类型的变量

```
