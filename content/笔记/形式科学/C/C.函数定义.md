##### 函数定义
- 函数定义
	- **函数定义**提供了[[C.函数|函数]]的具体实现, 将函数体[[C.复合语句|复合语句]]与[[C.函数声明|函数声明]]关联, 可以通过 [[C.return 语句|return]] 向其调用者返回一个值. 不同于[[C.函数声明|函数声明,]] 函数定义只允许在文件作用域
- 语法
	- `非指针声明符(形参列表) {函数体}`
		- 典型的 `S D(params) {}` 定义函数 `D` 接收参数 `params` 并运行函数体返回 `S` 
- 示例
```c
// 函数定义
int add(int a, int b) {
    return a + b;
}

void sayHello() {
    printf("Hello, World!\n");
}

```
