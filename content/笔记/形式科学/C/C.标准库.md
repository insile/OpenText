##### C.标准库
- 标准库
	- 标准库是所有符合 ISO C 标准的编译器内置[[C.头文件|头文件]]的集合, 为字符串处理, 数学计算, 输入/输出处理, 内存管理和其他一些操作系统服务等任务提供宏, 类型定义和函数

| 头文件                    | 描述                |
| ---------------------- | ----------------- |
| assert.h               | 提供断言功能，用于调试和错误检测  |
| complex.h (C99)        | 提供复数运算支持          |
| ctype.h                | 提供字符处理函数，如判断字符类型  |
| errno.h                | 提供错误码定义和错误处理      |
| fenv.h (C99)           | 提供浮点环境控制          |
| float.h                | 定义浮点数相关的限制        |
| inttypes.h (C99)       | 提供整数类型和格式化输出支持    |
| iso646.h (C95)         | 提供ISO 646替代符号的宏定义 |
| limits.h               | 定义数据类型的限制         |
| locale.h               | 提供本地化支持           |
| math.h                 | 提供数学函数            |
| setjmp.h               | 提供非本地跳转支持         |
| signal.h               | 提供信号处理功能          |
| stdalign.h (C11)       | 提供对齐功能            |
| stdarg.h               | 提供变长参数列表支持        |
| stdatomic.h (C11)      | 提供原子操作支持          |
| stdbit.h (C23)         | 提供位操作支持           |
| stdbool.h (C99)        | 提供布尔类型和值          |
| stdckdint.h (C23)      | 提供安全整数计算支持        |
| stddef.h               | 提供标准类型和宏定义        |
| stdint.h (C99)         | 提供定长整数类型          |
| [[C.stdio.h\|stdio.h]] | 提供标准输入输出函数        |
| stdlib.h               | 提供通用工具函数，如内存分配和转换 |
| stdnoreturn.h (C11)    | 提供 no-return 函数支持 |
| string.h               | 提供字符串操作函数         |
| tgmath.h (C99)         | 提供泛型数学函数          |
| threads.h (C11)        | 提供线程支持            |
| time.h                 | 提供时间和日期函数         |
| uchar.h (C11)          | 提供 Unicode 支持     |
| wchar.h (C95)          | 提供宽字符和多字节字符串支持    |
| wctype.h (C95)         | 提供宽字符分类函数         |
