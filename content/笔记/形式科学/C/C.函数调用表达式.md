##### 函数调用表达式
- 函数调用表达式
	- **函数调用表达式**是指在程序中通过函数调用[[C.后缀运算符|后缀运算符]] `()` 执行一个[[函数]]的[[C.表达式|表达式]], 完成特定的任务或操作. 在调用函数时, 将实参传递给函数形参, 执行完毕后可以返回一个值给调用者, 也可以没有返回值. 函数可以嵌套调用, 作为其他函数的参数, 返回值直接传递给另一个函数
- 示例
```c
#include <stdio.h>

int square(int x) {  
    return x * x;  
}  
  
int main() {  
    printf("Square of 4: %d\n", square(4)); 
    return 0;  
}
```
