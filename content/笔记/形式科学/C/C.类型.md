##### 类型
- 类型
	- **类型**是 C 语言的基础组成部分, 用于描述变量, 表达式和函数的属性以及如何进行内存管理和操作. C 是静态类型语言, 变量的类型在编译时确定, 不能动态更改. C 允许不同类型之间进行[[C.隐式转换|隐式转换]], 还可使用[[C.转型表达式|转型表达式]]. 派生类型是建立在基本类型的基础上, 无法简单地声明一个派生类型. 所有非函数非空的类型称为[[C.对象|对象]]
