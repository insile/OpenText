##### 数组下标表达式
- 数组下标表达式
	- **数组下标表达式**是指在程序中通过数组下标[[C.后缀运算符|后缀运算符]] `[]` 访问[[C.数组|数组]]元素的[[C.表达式|表达式]]
- 示例
```c
#include <stdio.h>

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    // 嵌套循环访问二维数组
    for (int i = 0; i < 2; i++) {         // 遍历行
        for (int j = 0; j < 3; j++) {     // 遍历列
            printf("matrix[%d][%d] = %d\n", i, j, matrix[i][j]);
        }
    }

    return 0;
}

```
