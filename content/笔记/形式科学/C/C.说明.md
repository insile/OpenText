 ##### C 说明
- C 说明
	- C 语言是一种通用的, 过程式的编程语言, 由丹尼斯·里奇在20世纪70年代初在贝尔实验室开发. C 语言因其高效性和灵活性, 成为操作系统开发, 系统编程, 嵌入式系统等领域的首选语言之一
- C 文档
	- C Primer Plus (6th Edition) - Stephen Prata
	- https://en.cppreference.com/w/
- C 标准
	- 1989 C89 ANSI C 标准
	- 1990 C90 ANSI C 标准被采纳为 ISO C 标准
	- 1995 C95 
	- 1999 C99
	- 2011 C11
	- 2017 C17
	- 2023 C23
