##### 语句
- 语句
	- **语句**是 C 程序的基本构建块, 是按顺序执行的 C 程序片段, 一条语句相当于一条完整的计算机指令. C 语句由标记, 表达式和其他语句组成. 构成另一个语句的组成部分的语句称为封闭语句的 “体”. 在 C 中, 大部分语句都以分号结尾
