##### 数字类型
- 数字类型 `Number`
	- 数字类型是用于表示数值的内置数据类型, Python 支持多种数字类型, 包括整数, 布尔数, 浮点数, 复数. 整数是没有小数部分的数字, 布尔类型也是整数. 浮点数是带有小数部分的数字, 用于表示实数, 它们在内存中通常是以 IEEE 754 标准的双精度浮点数表示的, 运算存在不确定尾数, 可用科学计数法表示, $x_1ex_2=x_1*10^{x_2}$. 复数是由实部和虚部组成的数字 `-a+bj`, 实部 `a`, 虚部 `b`, `.real`获得实部, `.imag`获得虚部
