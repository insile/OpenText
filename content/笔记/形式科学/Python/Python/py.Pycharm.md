##### Pycharm
- PyCharm
	- PyCharm 是 JetBrains 公司开发的一款强大的 Python 集成开发环境 (IDE). 它为 Python 开发者提供了许多功能, 以提高开发效率和代码质量
- Pycharm 添加解释器
	- 添加本地解释器
	- 从 Conda 环境添加
	- 定位 conda.bat 可执行文件, 并加载环境
	- 使用现有环境或创造新环境
- Python 控制台
	- PyCharm 的 "Python 控制台" 是一个交互式环境, 允许你直接输入和执行 Python 代码. 这是一个强大的工具, 用于快速测试代码片段、调试问题, 以及进行与项目相关的实时探索和交互, 允许打开多个控制台
- Pycharm 运行
	- PyCharm 提供了两种主要的运行方式：直接运行和在控制台运行, 直接运行后结束, 在控制台运行后可继续交互调试
- PyCharm 调试
	- 在代码中设置断点： 在你认为可能有问题的代码行左侧单击, 设置断点. 断点是调试过程中暂停执行代码的位置
	- 触发断点： 当程序运行到设置的断点时, 程序将会在那个位置暂停. 此时, 你可以查看变量、执行代码、单步执行等
	- 调试控制台： PyCharm 提供了调试控制台, 其中包含用于调试的各种工具. 你可以在这里查看变量值、控制程序的执行流程等
		- 查看调用栈： 在调试模式下, 你可以查看当前的调用栈, 了解代码的调用关系
		- 监视变量： 通过添加变量监视器, 你可以在调试过程中实时查看特定变量的值变化
		- 步过：在当前行上执行代码, 如果当前行有函数调用, 不会进入该函数, 而是一次性执行完
		- 步入：进入当前行执行的函数或方法内部. 如果当前行没有函数调用, 与步过相同
		- 单步执行我的代码：从当前位置继续执行程序, 直到下一个断点或程序结束
		- 步出：在当前函数或方法内执行完剩余的代码后, 跳出函数或方法, 停在调用函数的下一行
