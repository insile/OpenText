##### 库
- 库
	- 库是扩充 Python 程序功能的方式, 是非正式的称谓, 指代可供导入和使用的代码集合, 包括[[py.模块|模块]]和[[py.包|包]]等, 可通过[[py.导入系统|导入系统]]导入创建为一个[[py.模块类型|模块对象]]使用

```python
import fibo  # 自己的文件
fibo
# <module 'fibo' from 'D:\\python\\fibo.py'>

import numpy  # numpy 包
numpy
# <module 'numpy' from '...\\anaconda3\\envs\\lib\\site-packages\\numpy\\__init__.py'>
```
