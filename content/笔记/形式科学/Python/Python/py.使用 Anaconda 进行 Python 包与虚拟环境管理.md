##### Anaconda
- Anaconda
	- Anaconda 是 Python 的发行版, 旨在简化包管理和部署, Anaconda 包是从远程 channel 下载的, 这些 channel 是包含 conda 包的目录的 URL. 需要了解一些[[py.常用 Conda 命令|常用 Conda 命令]], 可以对比一下和 pip 的[[py.conda 与 pip 对比|区别]]
- 步骤
	1. 安装 Anaconda
	2. Anaconda 默认创建使用  base 环境的 Python，预装了核心包和工具
	3. Anaconda 两种管理方式
		- Anaconda Navigator 图形界面
			- 从 Environments 点击创建管理环境与包
		- Anaconda Prompt 命令行界面
			- 输入 Conda 命令创建管理环境与包
