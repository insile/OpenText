##### 赋值语句
- 赋值语句
	- 赋值语句是由赋值符号 `=` 构成的一行代码, 用来给变量赋予新的数据值和数据类型, 可以[[py.可迭代拆包|可迭代拆包]]
```python
# 普通赋值
	a = 1
	
# 多重赋值
	a = b = 1

# 多元赋值
	a, b, c = 1, 2, 3
	
	ls = [1,2,3]
	a, b, c = ls
	tu = (1,2,3)
	a, b, c = tu
	
	x, y = y, x # 交换xy值	

# 增量赋值
	+=: 加法赋值，x += y 相当于 x = x + y
	-=: 减法赋值，x -= y 相当于 x = x - y
	*=: 乘法赋值，x *= y 相当于 x = x * y
	**=: 幂运算赋值，x **= y 相当于 x = x ** y
	/=: 除法赋值，x /= y 相当于 x = x / y
	//=: 整除赋值，x //= y 相当于 x = x // y
	%=: 取余赋值，x %= y 相当于 x = x % y
	&=: 按位与赋值，x &= y 相当于 x = x & y
	|=: 按位或赋值，x |= y 相当于 x = x | y
	^=: 按位异或赋值，x ^= y 相当于 x = x ^ y
    >> =: 右移赋值，x >>= y 相当于 x = x >> y
	<<=: 左移赋值，x <<= y 相当于 x = x << y
```
