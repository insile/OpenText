##### 数字字面值
- 数字字面值
	- [[py.数字类型|数字]][[py.字面值|字面值]]是指表示不同类型数字值的文本表达方式
```python
# 整数字面值
10  # 10
10_000_000  # 10000000
0b_1110_0101  # 229
0b10  # 2
0o10  # 8
0x10  # 16

# 浮点数字面值
3.14  # 3.14
10.  # 10.0
.001  # 0.001
1.23e-4  # 0.000123 
3.14_15_93  # 3.141593

# 虚数字面值
2+3j  # (2+3j)

# 布尔类型字面值
True
False
```
