##### 条件判断符
- 条件判断符
	- `<` 小于; `>` 大于; `<=` 小于等于; `>=` 大于等于; `==` 等于; `!=` 不等于
	- [[py.in|in]] 包含; [[py.is|is]] 相同; [[py.not|not]] 非; [[py.and|and]] 与; [[py.or|or]] 或
```python
# `<` 小于
1<2  # True

# `>` 大于
1>2  # False

# `<=` 小于等于
1<=2  # True

# `>=` 大于等于
1>=2  # False

# `==` 等于
1==2  # False

# `!=` 不等于
1!=2  # False
```
