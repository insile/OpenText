##### 迭代器对象
- 可迭代对象 `Iterable`
	- 实现 [object.\_\_iter\_\_()](py.object.__iter__().md) 
	- 可使用[[py.for 语句|for 语句]], 不可使用[[py.next()|next()]]
	- [[py.序列类型|序列类型]]、[[py.集合类型|集合类型]]、[[py.字典类型|字典类型]]等类型是可迭代对象，但不是迭代器
	- 可迭代对象是一个更通用的概念, 表示可以被迭代的对象, 它可以被转换为迭代器, 能够[[py.可迭代拆包|可迭代拆包]]
- 迭代器对象 `Iterator`
	- 实现 [object.\_\_next\_\_()](py.object.__next__().md) 
	- 可使用[[py.for 语句|for 语句]]和[[py.next()|next()]]
	- 可以通过[[py.iter()|iter()]]函数获得一个迭代器
