##### 类型操作符
- 类型操作符
	- 属性引用 `对象.属性`
	- 抽取 `容器[索引]`
	- 切片 `序列[索引:索引]`
	- 调用 `可调用对象(参数)`
```python
a = '123'
print(a.__class__)  # 这个属性记录了变量类型，<class 'str'>

n = [1, 2, 3, 4, 5]
print(n[2])  # 抽取，输出 3

t = "Hello, World!"
print(t[7:12])  # 切片，输出 'World'

sum([1,2,3])  # 调用函数，6
list('123')  # 调用类，['1', '2', '3']
list('123').pop()  # 调用类方法，'3'
```
