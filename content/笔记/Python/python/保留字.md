##### 保留字
- 被编程语言内部定义保留使用的标识符，用到再看
	- 类与函数
		- [[class]]，[[def]]，[[lambda]]
		- [[global]]，[[nonlocal]]
		- [[return]]，[[yield]]
		- [[await]]，[[async]]
		- [[with]]
	- 条件循环
		- [[if]]，[[elif]]，[[else]]
		- [[for]]，[[while]]，[[break]]，[[continue]]
	- 异常
		- [[try]]，[[except]]，[[finally]]
		- [[assert]]，[[raise]]
	- 包
		- [[import]]，[[from]]，[[as]]
	- 运算
		- [[and]]，[[or]]，[[not]]，[[is]]，[[in]]，[[del]]，[[pass]]
	- 类型
		- [[False]]，[[True]]，[[None]]
