# 词义
- 英：/ˌʌnɪnˈtelɪdʒəbl/； 美：/ˌʌnɪnˈtelɪdʒəbl/
- #词性/adj  难以理解的；难懂的
# 例句
- If a computer is given unintelligible data , it returns unintelligible results .

	- 懂一门第二外语会使你更有资格任职，而且会使你成为一名更好的旅行者，更不用说，当你的脑袋不慎撞到橱柜门的时候，还能为你提供一整套新创（且难以理解的）脏话来使用。

	- 他们所用的大量行话是外人听不懂的。
