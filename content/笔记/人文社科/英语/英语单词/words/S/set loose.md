##### 词义
- 美：/ set luːs /；
- #词性/v  释放，放开，解放
##### 例句
- The dog mustn't be set loose .
	- 这只狗不可以放开。
- Such crazed banthas are set loose in the desert.
	- 这样疯狂的班萨会被放归沙漠。
- Her death, like Daddy's, must have set loose contradictory feelings in Mother.
	- 她的过世和老爸的一样，准是在母亲心中产生了矛盾的情绪。
