##### 词义
- 英：/ set aʊt /；美：/ set aʊt /；
- #词性/v  安排和系统地呈现，标出（如设计），制定计划；详细陈述、描述或背诵；开始具有明确目的，着手；开始某种课程、旅程或职业
##### 例句
- She has set out to seduce Stephen.
	- 她已开始勾引斯蒂芬。
- When they set out they were well prepared.
	- 他们出发时有很充分的准备。
- He set out on the long walk home.
	- 他动身走很长的路回家了。
