# 词义
- 英：/ˈbrɔːdkɑːst/； 美：/ˈbrɔːdkæst/
- #词性/v  广播；散布，传播(信息等)；播送(电视或无线电节目)；参加广播 ( 或电视 ) 节目演出， 作广播 ( 或电视 ) 讲话，提供广播 ( 或电视 ) 节目
- #词性/n  广播节目；电视节目；<英>政党政治广播节目
- #词性/adj  播音的；撒播的；广泛散布的
- #词性/adv  经广播，通过广播；四散地
