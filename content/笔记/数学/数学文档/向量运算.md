##### 向量运算
-  $\pmb{a}=(a_1,a_2,...)$, $\pmb{b}=(b_1,b_2,...)$, $\pmb{c}=(c_1,c_2,...)$
- 向量空间运算
	- $V=\{\pmb{v_1},\pmb{v_2},\pmb{v_3},...\}$, $\pmb{a},\pmb{b},\pmb{c}\in V$, $m,n\in \mathbb{R}$
	- 向量加法
		- $\pmb{a}+\pmb{b}=(a_1+b_1,a_2+b_2,...)$
	- 标量乘法
		- $\lambda\pmb{a}=(\lambda{a_1},\lambda{a_2},...)$
	- 向量空间运算律
		- $\pmb{a}+\pmb{b}\in V$
		- $\pmb{a} + (\pmb{b} + \pmb{c}) = (\pmb{a} + \pmb{b}) + \pmb{c}$
		- $\pmb{a} + \pmb{b} = \pmb{b} + \pmb{a}$
		- 存在零向量 $\pmb{0}$, $\pmb{a} + \pmb{0} = \pmb{a}$
		- 存在负向量 $-\pmb{a}$, $\pmb{a} + (-\pmb{a}) = 0$
		- $m\pmb{a}\in V$
		- $m(\pmb{a} + \pmb{b}) = m\pmb{a} + m\pmb{b}$
		- $(m + n)\pmb{a} = m\pmb{a} + n\pmb{a}$
		- $m(n\pmb{a}) = (mn)\pmb{a}$
		- $1\pmb{a} = \pmb{a}$
- 内积空间运算
	- 向量内积
		- $\pmb{a}\cdot\pmb{b}=a_1b_1+a_2b_2+...$
		- $\pmb{a}\cdot \pmb{b}=||\pmb{a}||\cdot||\pmb{b}||\cdot \cos(\theta)$
	- 向量长度
		- $||\pmb{a}||=\sqrt{\pmb{a}\cdot\pmb{b}}$
	- 向量距离
		- ${\rm dist}(\pmb{a},\pmb{b})=||\pmb{a}-\pmb{b}||$
	- 内积空间运算律
		- $\pmb{a}\cdot\pmb{b}=\pmb{b}\cdot\pmb{a}$
		- $(\pmb{a}+\pmb{b})\cdot\pmb{c}=\pmb{a}\cdot\pmb{c}+\pmb{b}\cdot\pmb{c}$
		- $(c\pmb{a})\cdot\pmb{b}=c(\pmb{a}\cdot\pmb{b})=\pmb{a}\cdot(c\pmb{b})$
		- $\pmb{a}\cdot\pmb{a}\geq0\land\pmb{a}\cdot\pmb{a}=0\leftrightarrow\pmb{a}=0$
