##### 集合论基础
- 集合
	- 一个集合是一些互不相同的对象的聚集, 这些对象被称为集合的元素, 集合中的元素是互不相同的, 不考虑元素的顺序
- 集合表示方式
	- 列举法,  列举集合中的元素, $A=\{1, 2, 3\}$ 表示包含元素 $1、2、3$ 的集合 $A$
	- 描述法, 使用描述性条件定义集合中的元素, $B=\{x|x\ 是正整数\}$ 表示所有正整数的集合 $B$
- 特殊集合
	- 空集, 不包含任何元素的集合, 通常用符号 $\emptyset$ 或 $\{\}$ 表示
	- 全集, 包含所有可能元素的集合, 通常用符号 $U$ 表示
	- 子集, 如果集合 $A$ 的每个元素也是集合 $B$ 的元素, 那么 $A$ 是 $B$ 的子集, 空集是任何集合的子集, $A \subseteq B$
	- 真子集, 如果 $A$ 是 $B$ 的子集但不等于 $B$, 那么 $A$ 是 $B$ 的真子集, $A \subset B$
	- 幂集,  一个集合的幂集是指包含该集合所有子集的集合
	- 相等集合, 两个集合相等, 当且仅当它们具有相同的元素
- 集合运算
	- 并集, 表示两个或多个集合中的所有元素的集合, $A \cup B$ 
	- 交集, 表示同时属于两个或多个集合的元素的集合, $A \cap B$
	- 补集, 表示一个集合中不属于另一个集合的元素的集合, $A'$
- 集合基数
	- 基数, 一个集合中元素的数量。如果集合 $A$ 中有 $n$ 个元素, 其基数为 $|A| = n$
- 集合的基本定律
	- 交换律 
		- $A \cup B = B \cup A$
		- $A \cap B = B \cap A$
	- 结合律
		- $(A \cup B) \cup C = A \cup (B \cup C)$
		- $(A \cap B) \cap C = A \cap (B \cap C)$
	- 分配律
		- $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
