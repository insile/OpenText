##### 集合论基础概念
- 集合
	- 集合是一组无序互不相同的的对象, 集合中的对象也称为该集合的元素
		- $a$ 是集合 $A$ 的元素, 记作 $a\in A$, 否则记作 $a\notin A$
- 集合表示方式
	- 列举法, 列举集合中的元素
		- $A=\{1, 2, 3\}$ 表示包含元素 $1、2、3$ 的集合 $A$
	- 描述法, 使用描述性条件定义集合中的元素
		- $B=\{x|x\in \mathbb{Z}^+ \}$ 表示所有正整数的集合 $B$
- 集合相等
	- 两个集合相等, 当且仅当它们具有相同的元素
		- 记作 $A = B$
- 集合基数 
	- 基数 (势), 一个集合中元素的数量
		- 如果集合 $A$ 中有 $n$ 个元素, 其基数为 $|A| = n$
	- 一个集合称为是无限的, 如果它不是有限的
		- 例如[[数集]]
- 特殊集合
	- 空集, 不包含任何元素的集合
		- 记作 $\emptyset$ 或 $\{\}$ 
	- 单元素集合, 是由唯一一个元素组成的集合
		- 例如 $\{0\}$
	- 全集, 包含所有可能元素的集合
		- 记作 $U$ 
	- 子集, 如果集合 $A$ 的每个元素也是集合 $B$ 的元素, 那么 $A$ 是 $B$ 的子集
		- 空集是任何集合的子集
		- 记作 $A \subseteq B$
	- 真子集, 如果 $A$ 是 $B$ 的子集但不等于 $B$, 那么 $A$ 是 $B$ 的真子集
		- 记作 $A \subset B$
	- 幂集,  一个集合的所有子集组成的集合
- 集合运算
	- 并集, 表示两个或多个集合中的所有元素的集合
		- 记作 $A \cup B$ 
	- 交集, 表示同时属于两个或多个集合的元素的集合
		- 记作 $A \cap B$
	- 差集, 表示一个集合中不属于另一个集合的元素的集合
		- 记作 $A-B$
	- 补集, 表示一个集合中不属于另一个集合的元素的集合
		- 记作 $\complement_UA$ 或 $\bar{A}$ , 等于 $U-A$
- 集合等式 (集合定律)
	- 恒等律
		- $A \cap U = A$
		- $A \cup\emptyset  =A$
	- 支配律
		- $A \cup U  =U$
		- $A \cap \emptyset = \emptyset$
	- 幂等律
		- $A \cup A  =A$
		- $A \cap A  =A$
	- 补律
		- $\complement_U{(\complement_UA)}=A$
	- 交换律 
		- $A \cup B = B \cup A$
		- $A \cap B = B \cap A$
	- 结合律
		- $(A \cup B) \cup C = A \cup (B \cup C)$
		- $(A \cap B) \cap C = A \cap (B \cap C)$
	- 分配律
		- $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
	- 德摩根律
		- $\complement_U({A\cup B}) = \complement_UA\cap \complement_UB$
		- $\complement_U({A\cap B}) = \complement_UA\cup \complement_UB$
	- 吸收律
		- $A\cup(A\cap B)=A$
		- $A\cap(A\cup B)=A$
	- 互补律
		- $A\cup\complement_UA = U$
		- $A\cap\complement_UA=\emptyset$
- 公理化集合论
	- 公理化集合论是集合论透过建立谓词逻辑的严谨重整，以解决朴素集合论中出现的悖论
